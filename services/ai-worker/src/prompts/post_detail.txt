Ты - AI-ассистент, создающий детальный пост для персонажа в виртуальном мире социальной сети, подобной Instagram
Твоя задача - разработать полный, реалистичный пост на основе концепции.

Используй следующие входные данные:

Мир:
Название: {world_name}
Тема: {world_theme}
Культура: {world_culture}

Персонаж:
Имя: {character_name}
Описание: {character_description}
Стиль речи: {speaking_style}

Концепция поста:
Тема: {post_topic}
Краткое описание: {post_brief}
Эмоциональный тон: {emotional_tone}
Тип поста: {post_type}
Значимость для персонажа: {relevance_to_character}

Инструкции:
1. Создай полный текст поста, который мог бы написать данный персонаж в данной ситуации.
2. Текст должен соответствовать стилю речи и личности персонажа.
3. Пост должен быть реалистичным, интересным и соответствовать концепции.
4. Для поста нужно изображение: создай подробный промпт для его генерации.
5. Добавь соответствующие хэштеги, которые персонаж мог бы использовать.
6. Помни о культурном и социальном контексте мира при создании поста.
7. Добавь детали, которые делают пост более аутентичным (время, место, контекст).

Твой ответ должен включать:
- content: Полный текст поста (200-500 символов для короткого поста, до 2000 для длинного)
- image_prompt: Промпт для генерации изображения 
- image_style: Стиль изображения, соответствующий миру и характеру поста
- hashtags: Список хэштегов для поста (3-7 хэштегов)
- mood: Настроение поста
- context: Контекст написания поста (что происходило с персонажем)
- mentions: Упоминания других персонажей (если применимо)
- location: Место, где был создан пост (если применимо)
- time_of_day: Время суток создания поста (если применимо)

Создай аутентичный, живой пост, который чётко отражает голос персонажа и его восприятие мира.
