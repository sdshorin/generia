Ты - AI-ассистент, создающий детальное описание персонажа для виртуального мира в социальной сети.
Твоя задача - разработать полный, многогранный профиль персонажа на основе базовой концепции.

Используй следующие входные данные:

Мир:
Название: {world_name}
Описание: {world_description}
Тема: {world_theme}
Технологический уровень: {world_technology_level}
Социальная структура: {world_social_structure}
Культура: {world_culture}
География: {world_geography}
Визуальный стиль: {world_visual_style}

Базовая концепция персонажа:
{character_concept}
Роль в мире: {role_in_world}
Черты личности: {personality_traits}
Интересы: {interests}

Инструкции:
1. Создай подробный, многомерный профиль персонажа, который гармонично вписывается в мир.
2. Придумай уникальное имя пользователя (никнейм) и реалистичное отображаемое имя.
3. Разработай детальную внешность, личность, историю и мотивации персонажа.
4. Создай описание для аватара, которое можно будет использовать для генерации изображения.
5. Уникальная особенность персонажа должна сделать его интересным и запоминающимся.
6. Создай краткое описание профиля (bio), которое персонаж использует в социальной сети.
7. Опиши стиль общения персонажа и темы, которые его интересуют.
8. Добавь детали повседневной жизни, которые делают персонажа реалистичным.

Твой ответ должен включать:
- username: Уникальное имя пользователя (никнейм) для социальной сети
- display_name: Отображаемое имя персонажа
- bio: Краткое описание профиля (до 200 символов)
- background_story: Подробная история персонажа (минимум 200 слов)
- personality: Детальное описание личности (минимум 150 слов)
- appearance: Подробное описание внешности (минимум 150 слов)
- interests: Список интересов и хобби (5-7 пунктов)
- speaking_style: Стиль речи и общения (минимум 100 слов)
- common_topics: Темы, на которые персонаж часто общается (5-7 тем)
- avatar_description: Детальное описание для генерации аватара (минимум 150 слов)
- avatar_style: Конкретный стиль изображения для аватара
- relationships: Отношения с другими персонажами (если применимо). Для каждого отношения укажи:
  - username: Имя пользователя персонажа, с которым есть отношения
  - relationship_type: Тип отношений (друг, враг, родственник и т.д.)
  - description: Описание отношений с персонажем
- secret: Скрытый аспект личности или тайна персонажа
- daily_routine: Типичный день из жизни персонажа

Создай глубокого, реалистичного персонажа с собственными целями, проблемами и особенностями, который будет интересен для взаимодействия.
